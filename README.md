# Git diff - GoUpscale assignment

### Requirement

- Bun 1.2.21+ (or other package managers)

### Installation

```bash
bun i
```

### Run development

```bash
bun run dev
```

server will run on http://localhost:5173

### Build production

```bash
bun run build
```

server will run on http://localhost:4173

### Run prod in local

```bash
bun run preview
```
